基础DOM 和CSS 操作

#元素内容的获取和修改

  1. html() 获取元素中HTML 内容
    html(value) 设置元素中HTML 内容
  2. text() 获取元素中文本内容
    text(value) 设置原生中文本内容
  3. val() 获取表单中的文本内容
    val(value) 设置表单中的文本内容

html 和 text的区别

$(‘#box’).html(); //获取html 内容
$(‘#box’).text(); //获取文本内容，会自动清理html 标签
$(‘#box’).html(‘www.li.cc‘); //设置html 内容
$(‘#box’).text(‘www.li.cc‘); //设置文本内容，会自动转义html 标签

追加数据

我们使用html()或text()设置元素里的内容时，会清空原来的数据。而我们期
望能够追加数据的话，需要先获取原本的数据。

表单

如果元素是表单的话，jQuery 提供了val()方法进行获取或设置内部的文本数据。

如果想设置多个选项的选定状态，比如下拉列表、单选复选框等等，可以通过数组传递
操作。

#属性获取和修改
attr(key) 获取某个元素key 属性的属性值
attr(key, value) 设置某个元素key 属性的属性值
attr({key1:value2, key2:value2…}) 设置某个元素多个key 属性的属性值
attr(key, function (index, value) {}) 设置某个元素key 通过fn 来设置

#删除属性
$(‘div’).removeAttr(‘title’); //删除指定的属性
